# Sales-Analysis
This project analyzes the “Sales Case Study.csv” dataset, which contains daily sales information for a single product in a retail store. The analysis focuses on deriving key metrics and insights to better understand sales performance.

## Objectives

- Understand daily sales performance and pricing trends.

- Calculate key metrics such as price per unit, gross profit, and gross profit percentage.

- Evaluate product performance during promotional periods using Price Elasticity of Demand (PED).

- Identify additional insights, patterns, and opportunities for improvement.

## Analysis Steps
**1. Data Cleaning & Preparation**
- Checked for missing or inconsistent values.
- Derived new metrics such as sales price per unit and % gross profit.

**2. Exploratory Data Analysis (EDA)**
- Examined trends in sales, quantity sold, and profitability.
- Identified periods of high/low performance and promotional activity.

**3. Insights & Recommendations**
- Assessed demand response to price changes using PED.
- Highlighted additional insights, trends, anomalies, and improvement opportunities

**Tools & Technologies Used.**
- Miro – Project planning and flow mapping
- SQL – Data extraction and metric calculations
- Excel – Visualizations, KPIs, dashboards
- Canva – Final presentation design
